Challenges Associated with Big Data


There are many problems which are required to be resolved in proper manner for implementation of big data, which are discribed below (Desarkar and Das 2017).[2]


  • Nowadays 80% of data stores in unstructured manner, transformation of this unstructured data into the structured interpretable format is a major challenge for Big Data mining.
  • High volume data generation is another important aspect, which is required to be concerned for tackling Big Data. Processes and tools should be organised in adequate manner with which high volume of data set can be processed in reasonable amount of time.
  • Immediate analysis result is major issues in Big Data due to huge volume of data. Sometime real time (quick) result of analysis is required for reasonable decision making.
  • Issues related to security have to be handled in proper manner as data in Big Data comes from various type of sources. This can be done with implementation of techniques such as verification, encryption and key management, logging and watermarking techniques (Desarkar and Das 2017).[2]
